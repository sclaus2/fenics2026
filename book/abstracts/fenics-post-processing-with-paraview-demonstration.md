**Submission type:** Presentation

**Presenter:** Louis Gombert (Kitware)

This software demonstration showcases the open source post-processing application ParaView. We demonstrate how to use the ParaView Python interface to gain insight into simulation data efficiently and in parallel, at scale. Additionally, we explore in situ 3D data analysis using ParaView-Catalyst, to analyze FEniCS simulation results live, without writing data to disk. Finally, we show how ParaView and its Python interface can be leveraged to export images and animations, to communicate scientific results to a large audience.
